We study properties of theories of n-spherical orders Kn which naturally generalize linear orders K2 and circular orders K3 .

A n-spherical order relation, for n ≥ 2, is described by a n-ary relation Kn satisfying the following conditions: (nso1) ∀x1, …, xn(Kn(x1, x2, …, xn) → Kn(x2, …, xn, x1)); (nso2) $\forall x_1,\ldots,x_n \biggm((K_n(x_1,\ldots,x_i,\ldots,x_j,\ldots,x_n)\land$ Kn(x1, …, xj, …, xi, …, xn)) $\leftrightarrow\bigvee\limits_{1\leq k<l\leq n} x_k\approx x_l\biggm)$ for any 1 ≤ i < j ≤ n; (nso3) $\forall x_1,\ldots,x_n\biggm(K_n(x_1,\ldots,x_n)\to$ $\forall t\left(\bigvee\limits_{i=1}^nK_n(x_1,\ldots,x_{i-1},t,x_{i+1},\ldots,x_n)\right)\biggm);$ (nso4) ∀x1, …, xn(Kn(x1, …, xi, …, xj, …, xn)∨  ∨ Kn(x1, …, xj, …, xi, …, xn)),  1 ≤ i < j ≤ n.

Structures 𝒜 = ⟨A, Kn⟩ with n-spherical orders are called n-spherical orders, too.

A n-spherical order Kn is called dense if it contains at least two elements and for each (a1, a2, a3, …, an) ∈ Kn with a1 ≠ a2 there is b ∉ {a1, a2, …, an} with  ⊨ Kn(a1, b, a3, …, an) ∧ Kn(b, a2, a3, …, an).

[th_dense_isom] If 𝒜 and ℬ are countable dense n-spherical orders, n ≥ 2, without endpoints for n = 2, then 𝒜 ≃ ℬ.

[th_dec] For any natural n ≥ 2 the theory Tn of dense n-spherical order is decidable.
