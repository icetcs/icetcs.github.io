The ℕaproche natural language proof checker allows proof checked mathematical formalizations that read like ordinary mathematical texts . The easiest way to use the ℕaproche system is to install the well-known Isabelle prover and to edit a file with a .ftl.tex suffix in the Isabelle/jEdit editor. Some example files are included with the Isabelle distribution.

In my talk I shall describe a ℕaproche formalization of the appendix of John L. Kelley’s General Topology . The appendix is a short introduction to Kelley-Morse set theory which is taken as the foundation for the book and which is developed up to some initial results on ordinals and cardinals. Kelley’s text lends itself to formalizations due to its formalistic and complete style of writing. Indeed many statements of the appendix can be transferred almost verbatim to ℕaproche. Considering, e.g., Kelley’s

38 THEOREM If x is a set, then 2x is a set, and for each y, y ⊂ x iff y ε 2x.

about the power class 2x of x, the obvious LaTeX source is a legitimate ℕaproche input, save for the commas:

\begin{theorem}
If $x$ is a set then $2^x$ is a set and for each $y$ $y \subset x$ 
iff $y \in 2^x$.
\end{theorem}

Based on the Kelley formalization I shall conclude with general remarks on the natural language of mathematics.
